# Loan Approval Prediction

This project focuses on predicting loan approval status using machine learning techniques.

## Model
A Support Vector Machine (SVM) classifier with a linear kernel was used for prediction.

## Workflow
- Data cleaning and preprocessing
- Handling missing values
- Feature encoding
- Model training using SVM
- Model evaluation

## Results
The model achieved an accuracy of 81% on the test dataset.

## Goal
The goal of this project is to build a simple and effective classifier for loan approval prediction.
